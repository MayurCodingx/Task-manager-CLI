# CLI Task Manager (Python)

A simple yet powerful **command-line task manager** built using Python.
This project allows you to manage daily tasks efficiently with features like priorities, deadlines, and persistent storage.

---

## 🚀 Features

* ✅ Add tasks with optional deadlines
* ⭐ Set task priority (High / Medium / Low)
* 📋 View all active tasks
* ✔️ Mark tasks as completed
* 🗑️ Delete tasks
* 📦 Persistent storage using JSON
* 🔄 Automatically saves data between sessions

---

##  Tech Stack

* **Language:** Python
* **Storage:** JSON file system
* **Concepts Used:**

  * Object-Oriented Programming (OOP)
  * File handling
  * Input validation
  * CLI interaction

##  How to Run

1. Clone the repository:

```
git clone https://github.com/mayurcodingx/cli-task-manager.git
```

2. Navigate to the project folder:

```
cd cli-task-manager
```

3. Run the application:

```
python main.py
```

---

##  Usage

* Choose options from the menu to manage tasks
* Enter task details like name, priority, and deadline
* Use menu options to update or delete tasks
